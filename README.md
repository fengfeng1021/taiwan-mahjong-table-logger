# 台麻牌桌記錄

一個手機優先的台灣麻將牌桌記錄工具。用點選麻將牌取代手打文字，快速記錄手牌、牌河、吃碰槓，並一鍵複製成可貼給 AI 教練判斷的文字。

## 功能

- iPhone 13 Pro 尺寸優先的手機介面
- 開局選擇局牌與自己的座位
- 底部一次顯示全部 34 種牌：萬、筒、條、字牌不用切換
- 牌河模式：照實際出牌順序點牌，自動輪到下一家
- 手牌模式：快速加入或移除自己的手牌
- 吃、碰、槓記錄
- 吃牌會根據上一張棄牌列出合法搭子
- 碰、槓會根據上一張棄牌自動帶出牌種，並選擇是哪一家叫牌
- 快速撤回
- 一鍵複製目前桌況文字
- 本機自動保存進度

## 使用

直接開啟 `index.html` 即可使用。若部署到 GitHub Pages，入口也是根目錄的 `index.html`。

## 建議流程

1. 選擇局牌與自己的座位。
2. 按「開啟當前對局」。
3. 切到「手牌」模式，點入自己的手牌。
4. 切回「牌河」模式，照牌桌實際出牌順序點棄牌。
5. 有人吃碰槓時，在上一張棄牌後按對應按鈕。
6. 輪到自己時按「複製」，把文字貼給 AI 教練。

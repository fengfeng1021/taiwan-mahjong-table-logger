# 台麻即時教練

手機優先的台灣麻將即時輔助面板。它用本機演算法根據手牌、牌河、剩餘牌與他家進度，估算勝率、他家聽牌傾向、吃碰槓建議與輪到你時的推薦棄牌。

## 功能

- 開局只需選「我這局第幾個摸牌」
- 起手錄牌後進入實戰模式
- 輪到你時，直接點摸到的牌，APP 會加入手牌並立即推薦要打哪張
- 其他人打牌後，若你能吃碰槓，APP 會顯示叫牌建議與理由
- 即時估算勝率、進張、距離和棄牌危險度
- 依牌河進度估算他家聽牌概率
- 手牌中推薦棄牌會高亮，點手牌即可打出
- 一鍵撤回、一鍵複製目前局面
- 本機自動保存進度

## 使用

直接開啟 `index.html`，或使用 GitHub Pages。

1. 選擇你這局第幾個摸牌。
2. 按「開局錄手牌」，點入起手牌。
3. 按「開始實戰」。
4. 輪到別人時，點他打出的牌。
5. 如果 APP 顯示可吃碰槓，按「過」或「照建議」。
6. 輪到你時，先點你摸到的牌，再點手牌裡要打出的牌。
